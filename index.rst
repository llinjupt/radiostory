收音机的故事
=========================

午夜的收音机又在耳边响起......

从CD机或者手机里听到的声音叫好听，但是从收音机里不经意听到的声音叫动听，带着童年时光的温度，穿过时空、越过田野和城市，通过一个神奇的小盒子蹦蹦跳跳地进了你的耳朵。

80年代出生的人，正好赶上收音机辉煌时代的尾巴，童年里拨弄着收音机，听儿歌和评书，同时也免不了父母的呵斥。父辈的三转一响已经远去，而我们早已长大，我们的世界再也没有收音机的位置，关于收音机的故事也真正成为了故事。而好故事需要好听众，你呢，已经长大的你还会是一个好听众吗？

“这是那一辈人留下的足迹。几场风雨后就要抹去了痕迹。这片土地曾让我泪流不止，它埋葬了多少人心酸的往事。” 

一个80后将那些散落在网络空间里的关于收音机的故事收集起来说给你听，同时向那些终将逝去的岁月和辛苦养育我们长大的父辈一代致敬，你们的痕迹不会那么被轻易抹去。

如果你愿意完善它，可以通过 git clone https://github.com/llinjupt/radiostory.git 获取它的源码，并通过 lli_njupt@163.com 联系我获取更新权限，让我们在某一个安静的午后再次打开收音机，一起追忆似水流年！
